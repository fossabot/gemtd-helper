# gemtd-helper

Warcraft 3 GEM TD Helper

### Building from source

#### Prerequisites

1. [Git](https://git-scm.com/downloads) version control
2. Text editor like [Visual Studio Code](https://code.visualstudio.com/)
3. [Node.JS](https://nodejs.org/en/) runtime

#### Build

```
npm install
```

#### Run

```
npm start
```
